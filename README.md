
# Atividade Separação de Sílabas
### Esse projeto foi desenvolvido para auxilar crianças do 2º ano do Ensino Fundamental I . Juntei a informática com a educação para assim ajudar as crianças na alfabetização e treinar minhas habilidades com HTML, Css e Bootstrap.

<h4 align="center"> 
🚀 Finalizado
</h4>

### 🏁 Instalação

Antes de começar, é bom ter um editor para trabalhar com o código como [VSCode](https://code.visualstudio.com/)
Clone este repositório usando git clone: https://github.com/Leticiafrnc/Atividade-silabas.git ou clique em Code/Download ZIP. Clique no arquivo "index.HTML" e veja a aplicação.

### 🛠 Tecnologias

As seguintes ferramentas foram usadas na construção do projeto:
- HTML
- Css
- Bootstrap v5.0
